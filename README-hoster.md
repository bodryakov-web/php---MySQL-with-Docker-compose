# Развертывание на hoster.kz

## Шаги по развертыванию

1. **Загрузите файлы на сервер:**
   - Загрузите `index.php` в корневую директорию вашего сайта (обычно `public_html` или аналогичную).

2. **Импортируйте базу данных:**
   - Используйте phpMyAdmin или другой инструмент управления БД на hoster.kz.
   - Выберите базу данных `p-351366_php-docker`.
   - Импортируйте файл `db-init-hoster.sql`. Это создаст таблицу `users` и вставит тестовые данные.

3. **Проверьте подключение:**
   - Откройте сайт в браузере. Если все настроено правильно, вы увидите форму для добавления пользователей и список существующих.

## Примечания
- База данных уже создана на хостинге с указанными параметрами.
- Если возникнут ошибки подключения, проверьте настройки MySQL на hoster.kz (хост: localhost, порт: 3306).
